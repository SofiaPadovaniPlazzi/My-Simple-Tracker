# My Simple Tracker

A local goal tracker for consistency, streaks, notes, exceptions, emotional status, and statistics.

## Run

```bash
python3 my_simple_tracker.py
```

The app runs locally in your browser. Your tracker data is stored in browser local storage.

## macOS App

A lightweight app wrapper is available at:

```text
mac-app/My Simple Tracker.app
```

Open it to start the local tracker server and launch the app in your browser.
